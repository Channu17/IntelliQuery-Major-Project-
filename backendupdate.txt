Visualization:
    ai/visualize/suggest , params = {results (query results array), query_context (optional)}
        - Uses Groq LLM to analyze data and suggest appropriate chart types
        - Returns ranked recommendations with reasons, x/y axis, grouping
        - Provides data insights and summary
    
    ai/visualize/generate , params = {results, chart_type, x_axis (optional), y_axis (optional), group_by (optional), customization (optional)}
        - Generates Plotly chart JSON for frontend rendering
        - Supported charts: line, bar, horizontal_bar, pie, scatter, histogram, box, area, stacked_bar, grouped_bar, heatmap, funnel, waterfall
        - Customization: colors, title, theme, height, width, show_legend, hovermode
        - Returns Plotly JSON format (render with plotly.js or react-plotly.js)


Autocomplete (Google-style):
    ai/autocomplete , params = {partial_query, datasource_id, limit (optional, default 5)}
        - Real-time query suggestions as user types
        - Uses Groq LLM (llama-3.3-70b-versatile) for fast, intelligent suggestions
        - Context-aware based on datasource schema
        - Intelligent caching (5 min TTL, LRU eviction)
        - Sub-second response time (<500ms avg)
        - Fallback suggestions if LLM unavailable
        - Returns: list of natural language query completions
        - Min 2 characters to trigger
    
    ai/autocomplete/cache/{datasource_id} , DELETE request
        - Clear autocomplete cache for a datasource
        - Use when schema changes or testing
